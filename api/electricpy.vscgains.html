electricpy.vscgains¶

electricpy.vscgains(Rs, Ls, tau=0.005, freq=60)¶

Voltage Sourced Converter Gains Calculator.

This function is designed to calculate the kp, ki, and omega-not-L values for a Phase-Lock-Loop based VSC.

Parameters

  • Rs (float) – The equiv-resistance (in ohms) of the VSC

  • Ls (float) – The equiv-inductance (in Henrys) of the VSC

  • tau (float, optional) – The desired time-constant, default=0.005

  • freq (float, optional) – The system frequency (in Hz), default=60

Returns

  • kp (float) – The Kp-Gain Value

  • ki (float) – The Ki-Gain Value

  • w0L (float) – The omega-not-L gain value
