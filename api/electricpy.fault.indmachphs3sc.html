electricpy.fault.indmachphs3sc¶

electricpy.fault.indmachphs3sc(t, Is0, Lr, Ls, Lm, Rr, Rs, wrf=0, freq=60, real=True)¶

Induction Machine 3-Phase SC Calculator.

Determines the short-circuit current at a specified time for a three-phase fault on an unloaded induction machine.

Parameters

  • t (array_like) – The time at which to find the current, may be int, float, or numpy array.

  • Is0 (complex) – The initial (t=0) current on the stator.

  • Lr (float) – Inductance of the Rotor (in Henrys).

  • Ls (float) – Inductance of the Stator (in Henrys).

  • Lm (float) – Inductance of the Magnetizing branch (in Henrys).

  • Rr (float) – Resistance of the Rotor (in Ohms).

  • Rs (float) – Resistance of the Stator (in Ohms).

  • wrf (float, optional) – Frequency (in radians/sec) of the rotor slip. default=0

  • freq (float, optional) – Base frequency of the system (in Hertz). default=60

  • real (bool, optional) – Control argument to force returned value to be real part only. default=True

Returns

ias – Fault Current

Return type

array_like
