electricpy.wireresistance¶

electricpy.wireresistance(length=None, diameter=None, rho=1.68e-08, R=None)¶

Wire Resistance Calculator.

Enter three values to calculate the remaing one. Even though every variable is unitless, please use the International System of Units.

\[R = \frac{\rho*l}{A}\]
Parameters

  • length ([float], optional) – Wire length, unitless

  • diameter ([float], optional) – Wire diameter, unitless.

  • rho ([float], optional) – Material resistivity, unitless Default value is copper resistivity: 16.8*10-9

  • R ([float], optional) – Wire resistance, unitless.

Returns

  • length ([float], optional) – Wire length, unitless

  • diameter ([float], optional) – Wire diameter, unitless.

  • rho ([float], optional) – Material resistivity, unitless Default value is copper resistivity: 16.8*10-9

  • R ([float], optional) – Wire resistance, unitless.
