electricpy.phasor.parallelz¶

electricpy.phasor.parallelz(*args)[source]¶

Parallel Impedance Calculator.

This function is designed to generate the total parallel impedance of a set (tuple) of impedances specified as real or complex values.

\[Z_{eq}=(\frac{1}{Z_1}+\frac{1}{Z_2}+\dots+\frac{1}{Z_n})^{-1}\]
Parameters

Z (tuple of complex) – The tupled input set of impedances, may be a tuple of any size greater than 2. May be real, complex, or a combination of the two.

Returns

Zp – The calculated parallel impedance of the input tuple.

Return type

complex
