electricpy.conversions¶

Description

electricpy Package - conversions Module.

>>> from electricpy import conversions

Filled with simple conversion functions to help manage unit conversions and the like, this module is very helpful to electrical engineers.

Built to support operations similar to Numpy and Scipy, this package is designed to aid in scientific calculations.
