electricpy.hp_to_watts¶

electricpy.hp_to_watts(hp)[source]¶

Horsepower to Watts Formula.

Calculates the power (in watts) given the horsepower.

\[P_{\text{watts}}=P_{\text{horsepower}}\cdot745.699872\]

Same as watts.

Parameters

hp (float) – The horspower to compute.

Returns

watts – The power in watts.

Return type

float
