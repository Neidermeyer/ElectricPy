electricpy.protor¶

electricpy.protor(Pem, slip)[source]¶

Rotor Power Calculator for Induction Machine.

Given the electromechanical power and the slip, this function will calculate the power related to the rotor (provided or consumed).

\[P_r=-\text{slip}\cdot\frac{P_{em}}{1-\text{slip}}\]
Parameters

  • Pem (float) – Electromechanical power in watts.

  • slip (float) – Slip factor in rad/sec.

Returns

Pr – Power related to the rotor in watts.

Return type

float
