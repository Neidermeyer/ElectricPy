electricpy.machslip¶

electricpy.machslip(mech, syn=60)¶

Machine Slip Calculator.

Given the two parameters (mechanical and synchronous speed, or frequency) this function will return the unitless slip of the rotating machine.

\[\text{slip}=\frac{\text{syn}-\text{mech}} {\text{syn}}\]
Parameters

  • mech (float) – The mechanical frequency (or speed), of the rotating machine.

  • syn (float, optional) – The synchronous frequency (or speed), defaults as a frequency set to 60Hz, default=60

Returns

slip – The rotating machine’s slip constant.

Return type

float
