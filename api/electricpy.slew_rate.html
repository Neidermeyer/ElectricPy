electricpy.slew_rate¶

electricpy.slew_rate(V=None, freq=None, SR=None, find='')[source]¶

Slew Rate Calculator.

This function is designed to calculate slew rate i.e the change of voltage per unit of time`

Parameters

  • V (float, optional) – Voltage, Volts; default=None

  • freq (float, optional) – Frequency, Hz; default=None

  • SR (float, optional) – Slew Rate, Volts/sec; default=None

  • find (str, optional) – Control argument to specify which value should be returned.

Returns

  • V (float) – Calculated Volatage

  • freq (float) – Calculated frequency

  • SR (float) – Calculated slew rate
