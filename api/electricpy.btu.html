electricpy.btu¶

electricpy.btu(kWh)¶

Killo-Watt-Hours to BTU Function.

Converts kWh (killo-Watt-hours) to BTU (British Thermal Units).

\[\begin{split}\\text{BTU} = \\text{kWh}\\cdot3412.14\end{split}\]

Same as btu.

Parameters

kWh (float) – The number of killo-Watt-hours

Returns

BTU – The number of British Thermal Units

Return type

float
