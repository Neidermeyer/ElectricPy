electricpy.xfmphs¶

electricpy.xfmphs(style='DY', shift=30)¶

Simple Transformer Phase-Shift Calculator.

Use with transformer orientation to evaluate the phase-shift across a transformer. For example, find the phase shift for a Delta-Wye transformer as seen from the delta side.

Parameters

  • style ({'DY','YD','DD','YY'}, optional) – String that denotes the transformer orientation. default=’DY’

  • shift (float, optional) – Transformer angle shift, default=30

Returns

phase – Phasor including the phase shift and positive or negative characteristic.

Return type

complex

Examples

>>> import electricpy as ep
>>> # Find shift of Delta-Wye Transformer w/ 30° shift
>>> shift = ep.xfmphs(style="DY",shift=30)
>>> ep.cprint(shift)
1.0 ∠ 30.0°
